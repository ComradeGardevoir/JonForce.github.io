Basic Review Problems

1. Create an integer, x that is a random number between 2 and 10.

2. Create a second integer, y that is a random number between 11 and 50

3. Write a program that prints all of the numbers between x and y.

4. Write a program that adds all of the numbers between 0 and 100. (Including the number 100). It should print the result.

5. Write a program that draws a background color of blue if the mouse in on the lower half of the screen, and a background color of red if the
mouse is on the upper half of the screen.


Array List Review Problems

1. Create an ArrayList that stores Strings. Add the Strings "Moose", "Awesome", "Snake"
to the list.

2. Create an ArrayList that can store Integers. Next, add all of the numbers
between 0 and x to the ArrayList (where x is a random number between 10 and 50).

3. Next, write a program that prints all of the numbers in the list you just created.

4. Create an ArrayList that stores Inegers. Add all of the integers between 0 and 50 to the list.
Next, remove all of the integers between 10 and 20 from the list.
Finally, write code that prints the list.

5. Create a class called Color. Color should have a R, G, and B integer values.
Next, create an ArrayList that stores Colors and add 10 random Colors to it.

6. [This one is hard] Given the ArrayList of random colors in the last problem,
how can we select a random color out of the list (and use it as our background color)?


Timining Problems

1. Write a program that prints out, "Tracer" every one second.

2. [This one is hard] Write a program that changes to a random background color
every 2 seconds

3. [This one is hard] Print "Hi" every draw() for one second, then "Bye" every draw() for one second
then reset.

Class Review Problems

1. Create a class called Circle. Circles have the following properties :
Three integers, r, g, and b, that represent their color.
Two floats, x, and y, that represent their position.

2. Create a constructor in the class. When the constructor is called, it should
set the x, y, r, g, and b to all be random values.

3. Create an ArrayList that can store Circles.

4. In the setup function, create one Circle and put it into the list.

5. In the draw function, write a loop that draws all of the circles in the list to the screen. Remember to use the Circles' fill colors.

6. [This part is hard] Write code so that every 1 second, it adds another Circle to the list.
